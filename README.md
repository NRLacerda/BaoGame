# BaoGame

BaoGame is a small Unity project created to learn and experiment with basic 2D platform game concepts.

The goal is not to build a complete production game yet, but to test common Unity workflows and gameplay systems in a simple project.

## Focus Areas

- 2D player movement
- Jumping and gravity
- Collision detection
- Basic level layout
- Projectiles
- Simple enemies or targets
- UI elements such as score, health, and menus
- Scene organization
- Unity project structure

## Project Type

- Engine: Unity
- Genre: 2D platformer
- Purpose: Learning, prototyping, and testing gameplay concepts

## Repository Structure

- `Assets/` - Game assets, scripts, scenes, prefabs, and other Unity content.
- `Packages/` - Unity package dependencies.
- `ProjectSettings/` - Unity project configuration.

## Notes

This project is expected to change frequently as new Unity concepts are tested.
